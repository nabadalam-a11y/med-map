ИНСТРУКЦИЯ — КАРТА РАНЕНИЙ

Назначение

Карта ранений предназначена для быстрой фиксации локализации травм, формирования медицинского отчёта и передачи информации старшему медику.

Инструмент рассчитан на использование в полевых условиях, работает офлайн и управляется одной рукой.

Выбор стороны тела

  • ПЕРЕД — передняя поверхность тела
  • СПИНА — задняя поверхность тела

Выбор стороны влияет только на отображаемую схему и список зон.

Выбор зоны ранения

Основной способ:

  • Тапните по номеру зоны на схеме тела
  • Зона подсветится жёлтым цветом
  • Зона появится в списке ниже

Альтернативный способ:

  • Введите номер зоны вручную
  • Нажмите «ДОБАВИТЬ / УБРАТЬ»

Повторный тап по зоне или повторный ввод номера снимает выбор.

ЛЕВО / ПРАВО

Для всех зон автоматически определяется сторона: левая или правая.

Автоопределение выполняется по положению зоны на схеме.

При необходимости сторону можно изменить вручную.

Исключения: позвоночная область и пупочная область (сторона не указывается).

Характер травмы

Для каждой выбранной зоны доступны следующие отметки:

  • Осколочное
  • СОР (слепое осколочное ранение)
  • МВТ (минно-взрывная травма)
  • Ожог
  • Перелом (только для конечностей)

Дополнительно, при необходимости:

  • Шина наложена
Логические ограничения встроены автоматически (переломы и шины не отображаются для головы, шеи и туловища).

Обязательные поля

Перед формированием отчёта необходимо заполнить:

  • Дата и время
  • Позывной
  • Комментарий (кратко)

Без заполнения обязательных полей:

  • копирование текста
  • отправка в Telegram
  • создание PDF

недоступны.

Отчёт

После выбора зон и заполнения полей можно:

  • Обновить текст отчёта
  • Скопировать отчёт в буфер обмена
  • Отправить отчёт через Telegram
  • Создать PDF (2 страницы: перед и спина)

Короткая сводка (для рации)

Кнопка «КОРОТКАЯ СВОДКА» формирует одну строку для быстрого устного доклада.

Пример:

СОР левая нижняя конечность (№35); МВТ правая верхняя конечность (№8)

Хранение данных

  • Все данные сохраняются локально на устройстве
  • Интернет не требуется
  • При перезагрузке страницы данные сохраняются

Не заменяет клиническое мышление.
Следовать инструкциям подразделения и старших медиков.
